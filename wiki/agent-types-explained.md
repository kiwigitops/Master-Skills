# Agent Types Explained

## Coding agents
Focus: editing code, running tests, and fixing issues.

## Research agents
Focus: collecting sources, summarizing evidence, and producing reports.

## Browser/computer-use agents
Focus: web navigation and UI interaction.

## Workflow/automation agents
Focus: event-driven processes, integrations, and operations tasks.

## Multi-agent systems
Focus: splitting responsibilities across cooperating agents.

## Voice/multimodal agents
Focus: text + audio + image/video input/output.

## When to use each

- If task is mostly code: start with coding agents.
- If task is mostly synthesis: use research agents.
- If task requires UI actions: use browser/computer-use agents.
- If task is repeatable operations: use workflow automation.
