# Learning Paths

## Path 1: Complete beginner (7 days)

Day 1: Read `wiki/beginners-guide.md`
Day 2: Read `wiki/agent-types-explained.md`
Day 3: Browse `indexes/beginner-friendly.md`
Day 4: Pick one framework entry and one coding agent entry
Day 5: Run one local test workflow
Day 6: Record limitations and questions
Day 7: Submit one documentation improvement PR

## Path 2: Builder (2 weeks)

Week 1:
- validate all entries locally
- inspect taxonomy and schema
- run one framework comparison

Week 2:
- add 2-3 new entries
- write one new skill
- generate indexes and open a PR

## Path 3: Maintainer (ongoing)

- schedule monthly review of stale entries
- enforce contribution checklist
- keep taxonomy stable, evolve only with rationale
- track docs quality and newcomer friction
