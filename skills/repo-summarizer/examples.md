# Repo Summarizer Examples

## Example: Coding agent project

Input:
- docs describe autonomous PR creation and test execution

Expected output:
- Summary focused on coding workflows
- Notes on required sandboxing and human review
- Tags such as `coding-agent`, `automation`, `repo-editing`
