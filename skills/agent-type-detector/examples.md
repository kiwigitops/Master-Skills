# Agent Type Detector Examples

## Example: Browser-first workflow

Signals:
- interacts with webpages
- clicks, types, screenshots

Result:
- primary: Browser / Computer Use Agents
- secondary: Tool Use / Integrations

## Example: Voice-first agent

Signals:
- real-time audio in/out
- low-latency call handling

Result:
- primary: Voice / Multimodal Agents
- secondary: Workflow / Automation
