# Repository Data Hub

This directory contains structured YAML entries for AI agent ecosystem projects.

## Folder layout

- `agent-frameworks/`
- `agent-sdks-libraries/`
- `skills-systems/`
- `coding-agents/`
- `browser-agents/`
- `research-agents/`
- `multimodal/`
- `multi-agent/`
- `evaluation/`
- `memory/`
- `tooling/`
- `automation/`

## Notes

- Folder names are browsing helpers.
- Canonical classification is always defined by each entry's `category` field.
- Validate all entry changes before opening a pull request.
- Upstream skill source manifests are stored separately in `data/upstream/`.
