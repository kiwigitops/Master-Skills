# Support

## Getting Help

Use one of these paths:

- Questions and usage guidance: open a GitHub issue
- Data or classification corrections: open a GitHub issue with references
- Security concerns: follow `SECURITY.md`

## Before Opening An Issue

- Read `README.md`
- Read `docs/how-to-use-this-repo.md`
- Validate your local changes with:
  - `python scripts/validate_repo_entries.py`
  - `python scripts/generate_indexes.py`
  - `python scripts/index_upstream_skills.py`

## Maintainer Scope

Maintainers can help with:

- taxonomy questions
- schema and validation issues
- contribution workflow and templates

Maintainers cannot provide:

- legal advice
- guarantees about third-party repositories listed in the catalog
