# Release v1.1.0

Date: 2026-04-08

## Summary

This release evolves Agent Ecosystem Atlas into a true repo-of-repos model for skills.

## Highlights

- Added federated upstream skill source tracking:
  - `data/upstream/skill-sources.yaml`
  - `indexes/upstream-skill-sources.md`
  - `scripts/index_upstream_skills.py`
- Added `Awesome Copilot` under `Skills Systems` catalog data.
- Expanded local operator skills from 5 to 6 with:
  - `upstream-skill-source-manager`
- Upgraded skill quality with explicit output formats and quality checks across all core skills.
- Added contributor tooling for upstream source additions:
  - `templates/upstream-skill-source-template.yaml`
- Added migration guidance:
  - `docs/adoption-prompt.md`
  - `docs/federated-skills.md`
- Polished README trust signals and conversion UX:
  - CI badge
  - last-updated badge
  - clearer star-value framing
- Enabled native GitHub Sponsors button support:
  - `.github/FUNDING.yml`

## Compatibility

- No breaking schema changes for existing repository entries.
- Existing validation and index scripts remain supported.

## Verification

- CI workflow runs:
  - `scripts/validate_repo_entries.py`
  - `scripts/generate_indexes.py`
  - `scripts/index_upstream_skills.py`
