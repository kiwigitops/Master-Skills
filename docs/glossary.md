# Glossary

## AI agent
Software that can plan, decide, and act toward a goal, usually by calling tools.

## Agent framework
A system for orchestrating agent behavior, tool calls, state, and execution flow.

## Agent SDK
A developer library used to build custom agent behavior.

## Skill system
A packaging pattern where instructions and optional resources are bundled into reusable task units.

## Tool use
An agent's ability to call APIs, CLIs, browsers, or other external capabilities.

## RAG
Retrieval-Augmented Generation. A pattern where models fetch external knowledge before generating outputs.

## Multi-agent system
A setup where multiple agents coordinate with distinct roles.

## Evaluation
A process for measuring model or agent quality, reliability, and regressions.

## Observability
Monitoring traces, logs, and metrics to understand agent behavior.

## Sandboxing
Running agent actions in constrained environments to reduce risk.

## Taxonomy
A fixed set of categories and rules used to classify entries consistently.
